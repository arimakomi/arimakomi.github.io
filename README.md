# استارلی مووی - وب‌سایت پخش ویدیو مشابه آپارات

## ویژگی‌ها
- پخش ویدیو با HTML5
- آپلود ویدیو + انتخاب ژانر و وضعیت
- امنیت پخش با `VideoStream`

## راه‌اندازی
1. دیتابیس MySQL ایجاد کنید
2. `config.php` را تنظیم کنید
3. فایل‌ها را روی سرور PHP آپلود کنید
